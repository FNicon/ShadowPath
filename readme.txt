/////Cara Menjalankan Game/////
Game dapat langsung dijalankan dengan cara menjalankan file.exe yang berada dalam direktori game dengan asumsi 
tidak ada file dalam direktori yang hilang. Game dijalankan dalam komputer yang menggunakan operating sistem 
windows yang menggunakan spesifikasi input/output berupa layar, keyboard, mouse, dan speaker.



/////Tutorial Game : Main Menu/////
Pertama kali menjalankan game, pemain akan dihadapkan dengan menu utama. Di dalam menu utama terdapat 4 tombol 
yaitu "New Game", "Load", "Options", dan "Quit"

New Game - memulai permainan dari level paling awal
Load - membuka panel yang menampilkan level yang akan diload dan jumlah soul yang sudah terambil. juga ada 
tombol untuk melakukan load
Options - membuka panel option untuk mengatur volume BGM dan SFX.
Quit - keluar dari permainan



/////Tutorial Game : Tutorial Level/////
Ketika pemain masuk pertama kali ke dalam permainan, pemain akan masuk ke dalam level tutorial. Level tutorial 
ini ada untuk mengenalkan pemain kepada lingkungan permainan dan juga kontrol dasar game. Pemain juga akan 
dipandu dengan tulisan2 yang akan memberi petunjuk yang akan membantu pemain.

Kontrol dasar yang bisa digunakan pemain dalam permainan adalah:
A/D - bergerak ke arah kiri dan kanan
W/S - naik dan turun tangga atau tali
Spasi - melompat
E - Masuk/Keluar dunia bayangan
Esc - mengeluarkan menu pause

Tombol W,A,S,D, dan spasi digunakan untuk bergerak selayaknya permainan platforming 2D pada umumnya. Selain itu 
tombol A dan D digunakan untuk menarik atau mendorong kotak jika pemain menemukannya dalam permainan. Untuk 
melepas pegangan ke kotak atau tali, pemain menggunakan tombol spasi

Dunia bayangan adalah fitur unik dari permainan ini. Dalam permainan, pemain bisa menekan tombol 'E' kapan saja 
untuk masuk atau keluar dari dunia bayangan. Dalam dunia bayangan, karakter pemain mendapatkan kemampuan baru 
untuk berinteraksi dengan bayangan layaknya benda asli. Dengan kemampuan ini, karakter pemain bisa berdiri di 
atas bayangan layaknya sebuah jembatan atau dinding. Kemampuan ini pula yang menjadi kunci dalam aspek puzzle 
permainan ini dimana pemain harus bisa memanipulasi bayangan agar bisa melewati rintangan.

menu pause bisa dimunculkan dengan menggunakan tombol ESC. Dalam menu pause terdapat pilihan pemain untuk 
melanjutkan permainan, memulai ulan level, membuka panel option. dan kembali ke menu utama
