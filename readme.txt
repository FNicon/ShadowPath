/////Cara Menjalankan Game/////
Game dapat langsung dijalankan dengan cara menjalankan file.exe yang berada dalam direktori game dengan asumsi tidak ada file dalam direktori yang hilang.
Game dijalankan dalam komputer yang menggunakan operating sistem windows yang menggunakan spesifikasi input/output berupa layar, keyboard, mouse, dan speaker.



/////Tutorial Game : Main Menu/////
Pertama kali menjalankan game, pemain akan dihadapkan dengan menu utama. Di dalam menu utama terdapat 4 tombol yaitu "New Game", "Load", "Options", dan "Quit"

New Game - memulai permainan dari level paling awal
Load - membuka panel yang menampilkan level yang akan diload dan jumlah soul yang sudah terambil. juga ada tombol untuk melakukan load
Options - membuka panel option untuk mengatur volume BGM dan SFX.
Quit - keluar dari permainan
